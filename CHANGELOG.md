# Changelog

All notable changes to `@ntatoud/styra` will be documented in this file.

## [0.1.0] - 2026-04-12

### Added

- `clsx`-like syntax and `cn` utility for merging class names ([#14](https://github.com/ntatoud/styra/pull/14))
- Support for function form of `className` prop ([#13](https://github.com/ntatoud/styra/pull/13))
- Boolean variant shorthand (e.g. `<Button loading />`)
- `VariantProps` helper type export
- MIT License
- Type-safe class variance builder as a maintained CVA replacement
- Benchmark package comparing batched CVA vs styra performance
- Published as `@ntatoud/styra` on npm
